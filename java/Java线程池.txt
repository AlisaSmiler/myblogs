线程状态

    shutdown--线程池中的线程如果是这种状态，代表不接收新任务，之前阻塞队列中的任务，它继续执行
    run -- 接收新任务，之前阻塞队列中的任务，也执行
    stop-- 停止状态 不接收新任务，也不处理老任务
    tidying -- 清理战场状态，有效线程数为0，代表所有任务都停止，并回收
    teminated -- 调用teminated() 终止状态,全部终止
